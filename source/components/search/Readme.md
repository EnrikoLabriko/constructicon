# Examples

**Standard Use**

Provides a standard form for querying a dataset.

```
<Search />
```

**Results**

You can pass through a result set to display.

```
<Search results={[
  {
    title: 'Cat',
    subtitle: 'Feline',
    image: 'http://placehold.it/100x100',
    url: 'http://www.example.com'
  }, {
    title: 'Dog',
    subtitle: 'Canine',
    image: 'http://placehold.it/100x100',
    url: 'http://www.example.com'
  }
]} />
```
