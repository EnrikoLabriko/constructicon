# Example Use

**Standard Use**

```
<MetricGroup>
  <Metric
    label='Target'
    amount='$500,000'
    icon='target'
  />
  <Metric
    label='Funds Raised'
    amount='$250,000'
    icon='dollar'
  />
  <Metric
    label='Supporters'
    amount='174'
    icon='star'
  />
</MetricGroup>
```

**Colors**

Set the background and/or foreground colors

```
<MetricGroup background='primary' foreground='light'>
  <Metric
    label='Target'
    amount='$500,000'
    icon='target'
  />
  <Metric
    label='Funds Raised'
    amount='$250,000'
    icon='dollar'
  />
  <Metric
    label='Supporters'
    amount='174'
    icon='star'
  />
</MetricGroup>
```
