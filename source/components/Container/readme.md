Site Container

```
<Container styles={{ backgroundColor: '#eee' }}>
  <div>Header</div>
  <div>Body</div>
  <div>Footer</div>
</Container>
```

```
<Container tag='section' styles={{ backgroundColor: '#eee' }}>

  <div>Footer</div>
</Container>
```
