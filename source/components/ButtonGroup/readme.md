Button Group

```
<ButtonGroup>
  <Button>Button 1</Button>
  <Button>Button 2</Button>
  <Button>Button 3</Button>
</ButtonGroup>
```

Adjust Padding

```
<ButtonGroup padding={{ x: 0.25, y: 1 }}>
  <Button>Button 1</Button>
  <Button>Button 2</Button>
  <Button>Button 3</Button>
</ButtonGroup>
```

Custom Styles

```
<ButtonGroup styles={{ backgroundColor: '#ddd' }}>
  <Button>Button 1</Button>
  <Button>Button 2</Button>
  <Button>Button 3</Button>
</ButtonGroup>
```
