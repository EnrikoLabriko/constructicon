Depends on [react-slick](https://github.com/akiran/react-slick). See [README](https://github.com/akiran/react-slick#readme) for full list of available options.

<br>

# Examples

**Standard Use**

```
<Carousel>
  <div>Slide 1</div>
  <div>Slide 2</div>
</Carousel>
```

**Autoplay**

```
<Carousel dots autoplay arrows={false}>
  <div>Slide 1</div>
  <div>Slide 2</div>
</Carousel>
```
